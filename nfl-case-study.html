NFL QB Touchdown Predictor

Machine Learning Case Study

Problem Statement

Predicting quarterback touchdown performance in NFL games is crucial for fantasy football, sports betting, and team strategy. Traditional analysis relies on basic statistics and expert intuition, but lacks the ability to process complex patterns across multiple variables simultaneously.

Challenge: Build a machine learning model that can accurately predict whether a quarterback will throw at least one touchdown in their next game using historical performance data and contextual factors.

Approach & Methodology

I developed an end-to-end machine learning pipeline using XGBoost, a gradient boosting algorithm known for its performance on tabular data and ability to handle feature interactions.

Data Collection & Preprocessing

  • Dataset: 2,500+ NFL games from 2018-2023
  • Features: 15 engineered features including recent performance metrics, opponent defense stats, weather conditions, and game context
  • Target: Binary classification (TD thrown: Yes/No)
  • Train/Val Split: 80/20 with temporal validation

Model Development

  • Algorithm: XGBoost with hyperparameter tuning
  • Feature Engineering: Rolling averages, opponent-adjusted stats, situational metrics
  • Validation: Time-series cross-validation to prevent data leakage
  • Calibration: Platt scaling for reliable probability estimates

Key Insights:

  • Recent passing yards per game (last 3 games) was the strongest predictor
  • Opponent pass defense ranking showed significant impact
  • Weather conditions (wind speed) affected accuracy more than expected
  • Home field advantage provided measurable boost

Technical Implementation

Deployment

Created a Streamlit web application that allows users to:

  • Upload quarterback statistics
  • Get instant touchdown predictions with confidence scores
  • View SHAP explanations for model decisions
  • Compare multiple quarterbacks side-by-side

What I'd Improve

  • Data Expansion: Include more contextual features like injury reports, team chemistry metrics
  • Model Ensemble: Combine XGBoost with neural networks for potentially better performance
  • Real-time Updates: Implement live data feeds for in-game predictions
  • Advanced Features: Add player-specific tendencies and situational awareness
  • Performance Monitoring: Set up model drift detection and retraining pipelines
